WORKFLOW THINKING :

Given a project brief → structured document:

Step 1: VALIDATE - Check minimum length, required keywords, data quality
Step 2: STRUCTURE - Parse brief, extract key entities (dates, budget, risks)
Step 3: PROCESS - Send to AI with strict JSON schema, temperature 0.3 for consistency
Step 4: VERIFY - Validate AI output has all required sections, fill missing with defaults
Step 5: GENERATE - Write to Word document with professional formatting
Step 6: DISTRIBUTE - Upload to OneDrive, create Asana task, add comment
Step 7: LOG - Save workflow state, errors, timing for audit
Step 8: HANDLE FAILURES - If AI fails → rule-based fallback, if API fails → retry, if partial → flag for human
